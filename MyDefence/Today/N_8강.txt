<오늘의 목표> - UI - UX
타워를 선택하는 UI 구성

<오늘의 과제>
1. 스크린 하단에 타워 선택 버튼을 2개 만든다

2. 스크린 하단의 첫번째 타워 선택 버튼 클릭 후에만  맵에 타워를 설치할 수 있다
- 버튼 클릭시  Debug.Log("머신건 타워를 선택 하였습니다!!"); 
1) 만약 저장된 타워가 없으면(선택버튼을 누르지 않으면) 설치 실패 처리하세요
  (선택버튼을 누르지 않으면 타워 설치를 실패해야 합니다.)
  선택버튼을 누르지 않고 타일을 선택하여 타워 설치하려고 하면 실패처리
  Debug.Log("타워를 설치하지 못했습니다.!!");
2) 첫번째 선택버튼 클릭 :  설치할 타워에 머신건 타워(프리팹)을 저장
3) 타일을 클릭하면 저장된 타워를 설치하세요
4) 만약 저장된 타워가 없으면(선택버튼을 누르지 않으면) 타일의 메털리얼이 변경되지 않게 하세요

3. 타일 위에 버튼(UI)이 있을 경우 타일 선택 또는 타워 설치가 안되게 하기

4. 머신건 타워의 프리팹 복사 AnotherTowerPrefab 만들고 두번째 버튼 연결해서 
 선택 기능, 설치 기능 구현
1)  두번째 선택버튼 클릭 : Debug.Log("다른 타워 선택 하였습니다!");
2)  두번째 선택버튼 클릭시 : AnotherTowerPrefab 저장
3)  맵타일 클릭하면 선택한 맵타일에 AnotherTower을 설치

<필요 스킬>
1. UI - Panel (Layout Group) , Button
2. EventSystem.current.IsPointerOverGameObject()

UITest - RectTransform
